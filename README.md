# PARSER DEMO

Программа для демонстрации работы парсера (синтаксический анализатор)

## Формат строки для разбора

`<name>|<date of berth>|<phone>`

+ `<name>` - имя
+ `<date of berth>` - дата рождения в формате `yyyy-mm-dd`
+ `<phone>` - номер телефона в международном формате

***Пример:***

`Риц Милана Аркадьевна| 2001-23-12|+79998887766`
